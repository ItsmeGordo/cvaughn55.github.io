C2F2L is a Cubing website that was forked and added upon when I noticed the original website had been offline for months and the creator inactive. It is a tool made by net13, and I added features such as New design, Different colored site themes, and Progrssive Web App functionality that allows the site to be installed on an Android or IOS device similar to a native app from the appstore. This allows for a fullscreen experience, and offline use.
